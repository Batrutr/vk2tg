# VK <-> Telegram бот для пересылки сообщений и медиа.

## Возможности

- Пересылка сообщений Telegram -> VK и VK -> Telegram
- Пересылка вложений (фото, документы, голосовые, видео, аудио)
- Поддержка ответов и пересланных сообщений из VK
- Персональный активный чат для каждого авторизованного Telegram пользователя
- Фильтрация входящих сообщений из VK только по чатам из `chats.json`
- Админ-команды для управления пользователями и перезагрузки конфигурации чатов

# Файлы конфигурации

## data.json

Хранит только рабочее состояние бота (без токенов).

Минимально необходимые поля:

```json
{
    "tg_ids": []
}
```

Дополнительно бот хранит служебные поля:

- `user_states` - персональные состояния пользователей (`current_chat`, `isChat`)
- `current_chat` и `isChat` - legacy-поля для совместимости

## .env

Секреты хранятся в переменных окружения.

Скопируйте шаблон и заполните:

```bash
cp .env.example .env
```

Поля в .env:

- TG_TOKEN - токен Telegram-бота
- VK_TOKEN - токен VK
- BOT_PASSWORD - опциональный пароль для команды /start
- TZ - часовой пояс

## chats.json

Формат:

- Личка: любое имя ключа без суффикса `_chat`
- Беседа: ключ с суффиксом `_chat`

Пример:

```json
{
    "nikita": "npmcursed",
    "work_chat": "242"
}
```

# Команды бота

```text
/start - Авторизация в боте (с паролем, если включен)
/help - Показать справку по командам
/status - Показать текущий чат, тип и вашу роль
/whoami - Показать ваш Telegram ID и роль
/chats - Показать список доступных чатов
/switch <имя> - Выбрать активный чат для отправки в VK
/mychat - Показать ваш выбранный чат и его ID
/clear_chat - Сбросить ваш активный чат
/allowed - Показать чаты, из которых принимаются входящие из VK
/users - Показать список авторизованных пользователей
/kick <id> - Удалить пользователя из доступа (только админ)
/reload_chats - Перезагрузить список чатов из файла (только админ)
```

# Локальный запуск

Перед запуском экспортируйте переменные окружения из .env.

1. Установите зависимости:

```bash
uv sync
```

2. Запустите бота:

```bash
set -a; source .env; set +a
uv run main.py
```

3. В Telegram отправьте боту `/start`.

# Docker

Сборка образа:

```bash
docker build -t vk-to-telegram-bot .
```

Запуск контейнера:

```bash
docker run --rm -it \
    -v "$(pwd)/data.json:/app/data.json" \
    -v "$(pwd)/chats.json:/app/chats.json" \
    -v "$(pwd)/media:/app/media:ro" \
    --name vk-to-telegram-bot \
    vk-to-telegram-bot
```

# Docker Compose

Запуск:

```bash
docker compose -f compose.yml up -d --build
```

Логи:

```bash
docker compose -f compose.yml logs -f
```

Остановка:

```bash
docker compose -f compose.yml down
```
